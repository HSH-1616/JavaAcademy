BOM : Browser에 대한 정보를 가지고 있는 객체들...

화면 : screen 객체
url : location 객체
브라우저정보 : navigator 객체
페이지이동이력 : history 객체

screen

client의 화면에 대한 정보를 저장하는 객체

navigator 객체

클라이언트의 위치정보를 가져오는 기능을 제공 -> 좌표

history

location

url창에 대한 정보를 관리하는 객체
페이지를 이동(요청)시키거나, 갱신하는 기능을 제공함.
페이지를 이동하는 방식
1. location.href속성을 변경하기 -> url주소
2. location.assign("url주소"); -> 이전페이지로 이동할수있음
3. location.replace("url주소"); -> 이전페이지로 이동을 못함.
4. location.reload(); -> 페이지를 새로고침 (F5, ctrl+r);
