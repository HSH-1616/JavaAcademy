//프로젝트 생성 후 실습!
// 사람들의 정보를 저장하는 프로그램을 생성
// 학생, 선생, 직원, 운전사를 저장하고 관리
// 학생 : 이름, 나이, 성별, 수강과목, 반
// 선생 : 이름, 나이, 성별, 전공과목, 담당과목
// 직원 : 이름, 나이, 성별, 부서, 전화번호
// 운전 : 이름, 나이, 성별, 담당지역, 운전경력
// 효율적인 클래스를 설계하고 만들어보자
// 각 객체를 한개씩 생성해서 내용 출력
// 학생 : 이름 나이 성별 반이 같으면 같은 객체
// 선생 : 이름 나이 전공과목이 같으면 같은 객체
// 직원 : 이름 나이 부서 가 같으면 같은 객체
// 운전 : 이름 나이 성별 담당지역 운전경력이 같으면 같은 객체
// 모두 변수로 필드데이터를 모두 출력할 수 있게 설정

// 성별은 '남','여' 만 입력이 가능하게 수정하고 
// age는 10이상만 가능, 이름은 두글자 이상만 가능 버전업!
