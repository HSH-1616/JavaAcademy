배열 활용하기

1. js의 배열은 java List와 유사하다.

2. 배열저장소에는 다양한 type의 값을 저장할 수 있다.

3. 인덱스 번호로 배열에 있는 값을 출력하고, 배열에 값을 저장할 수 잇다.

4. 길이제한이 없고, 자동으로 증가함 * 공간에 대해 미리 설정할 필요가 없다.

5. js배열은 Array오브젝트로 편의 메소드(함수)를 제공한다.

배열객체가 제공하는 함수 활용하기

indexOf("찾을값") : 매개변수로 전달된 값과 일치하는 요소를 찾아 인덱스번호를 반환

concat() : 두개이상의 배열을 합쳐주는 기능

join() : 배열을 문자열로 변경해주는 기능

reverse() : 배열의 순서를 반대로 변경해주는 기능

배열의 데이터를 조작하는 함수들...

배열의 값을 추가, 수정, 삭제하는 함수들

push() : 배열의 마지막 인덱스에 값을 추가하는 함수

pop() : 배열의 마지막 인덱스 값을 삭제하는 함수

shift() : 0번 인덱스의 값을 삭제하는 함수

unshift() : 0번 인덱스에 값을 추가하는 함수

slice() : 배열의 일부 인덱스를 복사해서 새로운 배열로 만들어주는 함수

slice(시작인덱스[,끝인덱스]);

splice() : 배열의 특정위치의 값을 삭제하고, 지정 위치에 새로운 값을 추가하는 기능

splice(시작인덱스, 지울갯수[,추가할값,추가할값....]);

includes() : 배열의 값중 매개변수와 일치하는 값이 있는지 확인 해주는 기능

배열에 있는 데이터를 순회해서 데이터를 조작하는 함수들...

find() : 배열에서 원하는 데이터 함개를 찾아서 반환해주는 기능 -> 단일값
filter() : 배열에서 원하는 다수의 데이터를 찾아서 반환해주는 기능 -> 배열
map() : 배열의 데이터를 원하는 방식으로 조작하여 배열로 반환하는 기능
forEach() : 배열의 모든 데이터를 순회하는 기능
reduce() : 배열에 특정값을 기준으로 누적시켜주는 함수
sort() : 배열을 정렬을 해주는 함수
* 위 함수들은 매개변수로 함수를 전달받아서 처리한다.
