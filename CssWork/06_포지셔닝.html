posiotion속성

태그를 페이지에서 워하는 위치에 배치할때 사용하는 속성
위치를 지정하는 속성과 같이 사용 : top, left, right, bottom
static : default값, 작성된 테그의 순서대로 배치하는 속성
absolute : 부모의 posiotion이 static일때 페이지의 왼쪽 상단 좌표가 0,0으로 설정
realative : 부모의 position이 static일때 부모의 왼쪽 상단 좌표가 0,0으로 설정
* 부모의 position이 absolute이나, realative로 설정이 되면 두 속성 모두 부모의 왼쪽상단 좌표가 0,0으로 설정
fixed : 태그의 위치를 특정 위치로 고정하는 속성
* 페이지를 스크롤해도 따라다님
